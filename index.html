Catatan untuk Kaka: Untuk mengunggah file audio robot Anda ke GitHub:

  1. Rekam suara robot yang mengatakan "Selamat datang di Kaka Music Studio"
  2. Simpan sebagai welcome-robot.mp3 di folder repository Anda
  3. Upload file tersebut ke GitHub bersama dengan file HTML ini
  4. Tombol play di atas akan memainkan suara robot Anda!
